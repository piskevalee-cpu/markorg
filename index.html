Your AI Terminal Assistant

MARK is a personal, persistent, and intelligent assistant that lives entirely in your terminal. Combines the power of state-of-the-art LLMs with a local, secure memory system.

Powerful Features

Local Memory

Remembers information when asked to with 100% local storage, then uses it to tailor its answers to your needs. No data brokers, no cloud storage of your personal facts.

Kleos System

Advanced prompt engineering homemade engine that analyzes and optimizes your prompts before execution for higher quality answers.

Secure & Private

API keys stored securely using your OS's native keychain. Your data stays on your machine.

Multi-Provider

Multi-Language

Terminal Native

Designed from the ground up for terminal use with beautiful, responsive UI.

Kleos System

Advanced Reasoning Engine

Kleos is MARK's advanced capabilities and prompt optimization engine. It transforms MARK from a simple chatbot into a reasoning agent.

How it Works

  1. Analyze: You provide a prompt. Kleos analyzes it for ambiguity, missing context, or hidden requirements and asks clarifying questions.
  2. Refine: It acts as an expert "Prompt Engineer", rewriting your query to be logically perfect.
  3. Execute: After approval, it sends the optimized prompt to the AI.

Result: Drastically higher quality and more detailed answers, better code, and deeper insights.

Usage

/kleos Explain how to build a neural network from scratch

Memory System

MARK's memory system is its standout feature. If you tell MARK to remember something, it can use that information to provide more personalized and context-aware responses.

How It Works

MARK uses a local SQLite database to store your memories. The system intelligently categorizes information and retrieves relevant memories when needed.

Privacy & Security

All memories are stored locally on your machine. No data is sent to external servers. Your personal information remains private.

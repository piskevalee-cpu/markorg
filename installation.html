Installation Guide

Get MARK up and running on your system in minutes with this step-by-step guide.

System Requirements

  • Python: Version 3.8 or higher
  • Operating System: Windows, macOS, or Linux
  • Internet Connection: Required for initial setup and API access
  • Disk Space: ~50MB for installation

Installation Steps

1

Verify Python Installation

First, ensure you have Python 3.8+ installed:

python --version
# or
python3 --version

If Python is not installed or you have an older version, download it from python.org.

2

Clone the Repository

Download the MARK source code from GitHub:

git clone https://github.com/piskevalee-cpu/MARK.git
cd mark

If you don't have Git installed, you can download the ZIP archive from the GitHub repository and extract it.

3

Install MARK

Install MARK globally on your system:

pip install .

Alternatively, you can install in development mode if you plan to contribute:

pip install -e .
4

Run MARK

Launch MARK for the first time:

mark

On first run, MARK will guide you through the initial setup process to configure your API keys.

Initial Setup Process

During the first run, MARK will guide you through the following setup steps:

1

User Name

Enter your name so MARK can address you personally.

2

AI Provider Selection

Choose between Groq (recommended for speed) or Google Gemini.

3

API Key Configuration

Enter your API key for the selected provider:

  • Groq: Get a key from console.groq.com
  • Google: Get a key from aistudio.google.com
4

Model Selection

Choose the AI model you want to use:

  • Groq Options: Llama 3.3 70B, GPT-OSS 120B, Kimi K2
  • Google Options: Gemini 2.5 Flash, Gemma 3 27B

Troubleshooting

Permission Errors

If you encounter permission errors during installation, try installing with the --user flag:

pip install --user .

Python Version Issues

If you have multiple Python versions, ensure you're using Python 3.8+:

python3.11 -m pip install .

Virtual Environment

For isolated installation, use a virtual environment:

# Create virtual environment
python -m venv mark_env

# Activate it (Windows)
mark_env\Scripts\activate
# Activate it (macOS/Linux)
source mark_env/bin/activate

# Install MARK
pip install .

Dependency Conflicts

If you encounter dependency conflicts, try upgrading pip first:

pip install --upgrade pip
pip install .

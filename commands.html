Basic Commands

Command Description Example
/help Show help menu with all available commands /help
/clear or /cls Clear the terminal screen /clear
/quit, /exit, /q Exit MARK /quit
/reset Clear current conversation context /reset
/stats View session usage and token costs /stats

AI Provider & Model Commands

Command Description Example
/model Switch AI provider and model interactively /model
/kleos <prompt> Activate Kleos System for advanced reasoning /kleos Explain quantum computing

Memory Commands

Command Description Example
/memory list View all saved memories /memory list
/memory search <term> Semantic search in memory /memory search Python
/memory delete <id> Delete a specific memory by ID /memory delete 5
/memory clear Delete all memories /memory clear

Natural Language Memory Commands

You can also use natural language to store memories:

English

remember that my name is John
remember that I prefer Python for scripting
remember that I live in New York
remember that my favorite programming language is Python

Italian

ricorda che mi chiamo Marco
ricorda che lavoro come sviluppatore
ricorda che vivo a Milano
ricorda che amo il caffè

Configuration Commands

Command Description Example
/config Update API keys and default settings /config
/changeusr <name> or /changename <name> Change your username /changeusr Alice

Command Usage Examples

Getting Started

# Get help
/help

# Clear the screen
/clear

# Exit MARK
/quit

Managing Memory

# Store information
remember that my preferred language is Python

# View stored memories
/memory list

# Search for specific memories
/memory search language

# Delete a memory
/memory delete 3

Using Kleos System

# Activate advanced reasoning
/kleos Explain how to build a neural network from scratch

# Kleos will analyze, ask questions, and provide detailed response

Switching Models

# Change AI provider/model
/model

# Follow the interactive prompts to select provider and model
